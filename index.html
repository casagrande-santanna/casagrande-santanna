Otimização da Barra de Abas

As abas agora têm melhor responsividade, espaçamento e feedback visual ao interagir.

Mudanças principais:

  • ✓ Padding vertical aumentado: 8px → 10px
  • ✓ Font size melhorado: 9px → 10px
  • ✓ Transição suave: transition: .2s
  • ✓ Efeito hover: Muda cor ao passar mouse
  • ✓ Ativa com font-weight: 600
.tab {
  padding: 10px 2px 8px; /* ← Melhorado */
  font-size: 10px; /* ← Aumentado */
  transition: .2s; /* ← Novo */
}

.tab:hover {
  color: var(--v); /* ← Novo: Feedback */
}

.tab.active {
  font-weight: 600; /* ← Novo: Destaque */
}
